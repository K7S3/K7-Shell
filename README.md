# K7_SHELL

Shell Written in C language. The filenames correspond to each of the builtin feature.

## COMPILE
```make```

## RUN
```./k7_shell```

## Features

* semicolon separated commands
* redirection
* piping
* Auto tab completion
* up and down arrow

## BUILTIN

* cd
* pwd
* echo
* ls
* pinfo
* history
* nightswatch (dirty)
* cronjob
* kill
* jobs
* set and unset var
* bg and fg
* kjob
* Control C and Control z
* quit
